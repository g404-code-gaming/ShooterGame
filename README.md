# ShooterGame
Pour cette 3ème partie de Unreal Engine, nous allons nous attarder sur le coeur de l'action d'un jeu FPS. 

Cette partie consiste à travailler les éléments suivants : 

### IA 

Nous allons créer et programmer des PNJ dangereux qui vont se déplacer, détecter et poursuivre le joueur. 

### Combat 

Nous allons ajouter les bases nécessaire pour que le joueur puisse combattre avec une arme à feu. Cela implique de pouvoir rammasser l'arme, tirer sur des ennemis, et subir soi-même des dégâts.
